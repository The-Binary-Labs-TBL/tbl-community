# The Binary Labs Community

This repository is the shared space for conversations around The Binary Labs projects.

Use this repository when you want to:

- ask questions about repositories or engineering decisions
- propose ideas before opening implementation work
- discuss architecture, tooling, or platform direction
- share feedback with maintainers and other contributors
- start broader community conversations that do not belong in a single code repository

## How to use this repository

The main purpose of `tbl-community` is GitHub Discussions.

- Start a discussion for questions, ideas, and open-ended topics
- Link to a specific repository when your topic relates to existing code
- Move to a pull request only once the discussion becomes actionable work

## What belongs here

Good topics for this repository:

- cross-repository ideas
- requests for feedback
- product or platform direction questions
- contributor onboarding questions
- community process discussions

Topics that usually belong elsewhere:

- bug reports with clear reproduction steps in a specific repository
- implementation work that is ready for a pull request
- private security issues, which should follow the security reporting process
